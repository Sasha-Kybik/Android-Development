# Sunset
    
    
**Описание**

Приложение показывает анимацию заката Солнца при нажатии на экран, и восход Солнца при повторном нажатии на экран с последующим изменением цвета фона неба.
    
    
**Приобретённые навыки**

• Знакомство с анимацией.
